## ⚡ Fun Facts
> 我几乎从来不用 AI 进行辅助开发或让它几乎直接参与我的任何项目代码 \
> 除非是万不得已的情况（临时精神、身体问题但是不想丢失我的概念而只能先编写简易的快速原型）下 \
> 我也会绝对确保在commit前把 AI 生成的所有代码都进行全面测试 \
> 并确保理解的万分透彻，让它就像是我自己写的代码一样在我脑中印象深刻且清晰 \
> 当然，事实上由 AI 生成的代码保留的并不是很多，除了极简单的代码(如编写EasingFunc)/自动猜测补全(如跨国语言字符串和注释等)，因为我总能在未经prompt限制的ai生产的代码中发现不符合我的要求的行为 \
> 面对陌生的负载时，我总是更倾向于引用人类文档，并有时会发给ai让其带我更快入门，但绝不是直接空手询问ai
